# 🏺 Deeper: The Game

**Deeper** to dynamiczna gra arcade typu *dungeon crawler* stworzona w silniku **Godot 4**. Wciel się w maga, unikaj strażników lochów i zbieraj skarby, by zapisać się na kartach historii!

## 🚀 Pobierz i zagraj (Wersja .exe)

Jeśli chcesz po prostu zagrać bez instalowania silnika Godot:
1. Pobierz pliki `gra.exe` oraz `gra.pck`.
2. **Ważne:** Oba pliki muszą znajdować się w tym samym folderze, aby gra mogła się uruchomić.
3. Uruchom `gra.exe` i ciesz się rozgrywką!

## ✨ Funkcje gry

* **Mechanika Uników:** Płynne sterowanie postacią maga w poszukiwaniu monet.
* **Inteligentny Wróg:** Przeciwnik (Wojownik), który nieustannie podąża Twoim śladem.
* **Globalny Ranking TOP 10:** Zintegrowany system zapisu wyników w formacie JSON (`user://ranking.save`).
* **Oprawa Audio:** Klimatyczne efekty dźwiękowe przy zbieraniu skarbów i przegranej.
* **Interfejs Użytkownika:** Menu główne, ekran pauzy (ESC) oraz system wpisywania imienia po zakończeniu gry.

## 🎮 Sterowanie

| Akcja | Klawisz |
| :--- | :--- |
| **Ruch** | Strzałki (Góra, Dół, Lewo, Prawo) |
| **Pauza/Menu** | ESC |
| **Szybki Restart** | R (na ekranie końca gry) |

## 🛠️ Specyfikacja Techniczna

* **Silnik:** Godot Engine 4.x
* **Język skryptowy:** GDScript
* **Zasoby:** Grafiki Pixel Art oraz dedykowane efekty dźwiękowe.

## 📦 Dla Twórców (Kod źródłowy)

Jeśli chcesz przejrzeć kod lub edytować projekt w silniku Godot:
1. Pobierz folder `gra` z tego repozytorium (możesz pobrać całe repozytorium jako ZIP przez przycisk **Code -> Download ZIP**).
2. Otwórz **Godot Engine 4**.
3. Wybierz opcję **Import** i wskaż plik `project.godot` znajdujący się wewnątrz folderu `gra`.
4. Po załadowaniu projektu naciśnij **F5**, aby przetestować grę!
